# Концепт 3 — Merge Shop (магазин + объединение предметов)

## Идея одним предложением
Объединяйте одинаковые предметы, чтобы открыть более редкие и продавать их покупателям.

## Для кого
Любители «собирать и апгрейдить», лёгкий менеджмент.

## Ядро
- Генератор создаёт предмет 1‑го уровня → объединяем 2× lvl1 = lvl2 → продаём → получаем монеты → улучшаем генератор/полки

## Управление
- Drag‑and‑drop, один экран

## Монетизация
- Rewarded: ускорить генератор/дать бесплатный предмет
- Interstitial: между волнами покупателей (редко)

## MVP объём (2 недели)
- 5 уровней предметов, генератор, 6 слотов инвентаря
- 2 типа покупателей, простая очередь, сейвы

## План (12 дней)
- Д1–Д2: поле/слоты, drag‑and‑drop, генератор
- Д3: merge‑логика уровней
- Д4: покупатели/очереди/продажи
- Д5: экономика/цены
- Д6: сейвы
- Д7: UI/анимации
- Д8: Yandex SDK (rewarded/interstitial)
- Д9: звук/иконка
- Д10: баланс «первые 30 минут»
- Д11: тесты/фиксы
- Д12: публикация

## Метрики‑решения (ориентиры)
- Средняя сессия ≥ 7–10 минут
- D1 (органика) ≥ 18–25%

## Риски
- Захламление поля → авто‑сжатие/кнопка «уборка»

## A/B идеи
- Скорость генератора, цена апгрейдов, редкость предметов
